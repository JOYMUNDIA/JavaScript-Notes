From Zero Knowledge → Confident JavaScript Developer

Step 1: Basic Math Operations

JavaScript can perform basic arithmetic using operators: 

let a = 10;
let b = 3;

console.log(a + b); // Addition → 13
console.log(a - b); // Subtraction → 7
console.log(a * b); // Multiplication → 30
console.log(a / b); // Division → 3.333333...
console.log(a % b); // Modulus → 1 (remainder)
console.log(a ** b); // Exponent → 1000 (10^3)

Step 2: Math Object Basics

JavaScript has a built-in Math object with useful functions for numbers.

Common Methods:

Math.round(4.6)   // 5 → rounds to nearest integer
Math.floor(4.9)   // 4 → rounds down
Math.ceil(4.1)    // 5 → rounds up
Math.abs(-7)      // 7 → absolute value
Math.pow(2, 3)    // 8 → 2^3
Math.sqrt(16)     // 4 → square root
Math.min(2, 5, 1) // 1 → smallest value
Math.max(2, 5, 1) // 5 → largest value
Math.random()     // 0 ≤ x < 1 → random decimal

- Combine methods to create more complex behavior. - Example: random integer between 1 and 10: 

let randomInt = Math.floor(Math.random() * 10) + 1;
console.log(randomInt);

Step 3: Using Math Constants

JavaScript provides useful constants through Math: 

console.log(Math.PI);   // 3.141592653589793
console.log(Math.E);    // 2.718281828459045
console.log(Math.LN2);  // 0.6931471805599453
console.log(Math.LN10); // 2.302585092994046
Mini Challenge: Calculate the circumference of a circle with radius 5 using Math.PI.

Step 4: Order of Operations

JavaScript follows the standard math rules (PEMDAS): 

let result = 2 + 3 * 4; // 14 → multiplication before addition
let result2 = (2 + 3) * 4; // 20 → parentheses change order

Always use parentheses if you want to be explicit.

Step 5: Mental Model – Math in JavaScript

Think of it like a calculator inside your program:

  • Numbers: inputs
  • Math operators: calculator buttons (+, -, *, /, %)
  • Math object: extra functions (ceil, floor, pow, sqrt, random)
  • Output: console.log() or variables

You can chain operations and functions to solve real-world problems.
