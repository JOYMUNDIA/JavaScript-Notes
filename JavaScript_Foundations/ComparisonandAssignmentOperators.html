From Zero Knowledge → Confident JavaScript Developer

Step 1: What Are Operators?

Think of operators as instructions for your variables. They tell JavaScript to perform a specific action on one or more values.

Just like math in real life, JavaScript has operators to:

  • Compare values
  • Do calculations
  • Combine logic
  • Store or update values
let a = 5;
let b = 10;
let sum = a + b; // + is the addition operator

Here, + tells JavaScript to add the numbers stored in a and b.

Step 2: Comparison Operators

Comparison operators check relationships between two values and always return a boolean (true or false).

Operator Meaning Example Result
== Equal (loose) 5 == "5" true
=== Equal (strict) 5 === "5" false
!= Not equal (loose) 5 != "6" true
!== Not equal (strict) 5 !== "5" true
> Greater than 10 > 5 true
< Less than 3 < 7 true
>= Greater than or equal 5 >= 5 true
<= Less than or equal 4 <= 5 true
Try It: Write expressions using ==, ===, !=, >, < and see what returns true or false.

Step 3: Logical Operators

Logical operators combine or invert boolean values. Think “yes/no” decisions in code.

Operator Meaning Example Result
&& AND → true if both sides are true true && false false
|| OR → true if at least one side is true true || false true
! NOT → inverts the boolean !true false 
let isAdult = true;
let hasID = false;

console.log(isAdult && hasID); // false
console.log(isAdult || hasID); // true
console.log(!isAdult);         // false

Step 4: Arithmetic Operators

Arithmetic operators do math in JavaScript. The basics are +, -, *, /, %, and **.

Operator Meaning Example Result
+ Addition 5 + 3 8
- Subtraction 10 - 4 6
* Multiplication 6 * 7 42
/ Division 12 / 3 4
% Remainder 10 % 3 1
** Exponentiation 2 ** 3 8
Try It: Calculate the remainder of 15 divided by 4. Then, raise 3 to the power of 4.

Step 5: Assignment Operators

Assignment operators store values in variables. They can also update values quickly.

Operator Meaning Example Result
= Assign value x = 5 x is 5
+= Add & assign x += 3 x = x + 3
-= Subtract & assign x -= 2 x = x - 2
*= Multiply & assign x *= 4 x = x * 4
/= Divide & assign x /= 2 x = x / 2
%= Remainder & assign x %= 3 x = x % 3 
let score = 10;
score += 5; // score = 15
score *= 2; // score = 30
Mini Challenge: Start with let points = 20; and update it using at least 3 different assignment operators.

Step 6: Combining Operators

Operators can work together in expressions. Parentheses () can help control order. 

let age = 20;
let hasLicense = true;

let canDrive = (age >= 18) && hasLicense;
console.log(canDrive); // true

Follow the order of operations: parentheses → exponents → multiplication/division → addition/subtraction → comparison → logical operators.

Mini Challenge: Write an expression to check if a person is an adult (age >= 18) and has enough money (money >= 10) to buy a ticket.

Step 7: Mental Model for Operators

  • Comparison: Compares two values → true/false.
  • Logical: Combines booleans → true/false.
  • Arithmetic: Does math → returns number.
  • Assignment: Stores or updates values.

Think of it like a factory: numbers and variables go in, operators perform actions, results come out.
